## Descrição:
Graças ao aumento da demanda por leitura e a popularização das compras online, as livrarias precisam de sistemas eficientes para disponibilizar seus catálogos de forma organizada, simples e acessível. Muitos sistemas atuais apresentam dificuldades na busca por livros, organização por categorias e no gerenciamento de estoque, o que pode impactar diretamente a experiência do consumidor.
Diante disso, este projeto propõe o desenvolvimento de um sistema de e-commerce voltado para a venda de livros, promovendo a acessiblidade a leitura de forma descomplicada, permitindo que os usuários pesquisem títulos, naveguem por categorias, adicionem produtos ao carrinho e realizem compras de forma simples e intuitiva.

## Objetivos principais
- Criar um e-commerce de livros organizado, facilitando a experiência do usuário;
- Implementar a listagem de livros por categorias, permitindo buscas eficientes para o catálogo;
- Garantir que o sistema controle a quantidade de livros, impedindo vendas acima do disponível;
- Estabelecer um sistema de login seguro com níveis de permissão distintos entre Clientes e Administradores;
- Garantir um processo fluido desde o cadastro inicial até a finalização do pagamento e geração do resumo do pedido;
- Implementar uma arquitetura robusta utilizando Programação Orientada a Objetos, buscando um código limpo e modular.

## Motivação
A motivação central deste projeto nasce da necessidade de unir a tecnologia ao incentivo à leitura de forma eficiente. Percebemos que, apesar do crescimento do mercado digital, muitos leitores ainda esbarram em plataformas confusas, com buscas ineficientes e falhas de estoque que acabam frustrando a experiência de compra. Queremos resolver essa dor criando um ambiente onde o foco do usuário seja apenas encontrar e aproveitar sua próxima leitura.
Além do impacto direto para o consumidor, este projeto nos motiva tecnicamente a tirar a teoria do papel. É a oportunidade perfeita para consolidar nossos conhecimentos em Programação Orientada a Objetos (POO), enfrentando o desafio de construir uma arquitetura de e-commerce que seja segura, limpa e modular desde a sua base.

## Dados Pincipais
O projeto consiste no desenvolvimento de um sistema de e-commerce de livros com foco em uma navegação simples e eficiente. Em termos de funcionalidades principais, o sistema permite a busca de livros, navegação por categorias, adição e remoção de itens no carrinho e finalização de pedidos, incluindo a geração de um resumo da compra. A ideia é garantir um fluxo direto, sem etapas desnecessárias, facilitando a interação do usuário com o sistema.

Nas regras de negócio, o ponto central é o controle de estoque, impedindo que produtos indisponíveis sejam vendidos, evitando inconsistências no sistema. Além disso, foi implementado um sistema de autenticação com dois níveis de acesso (cliente e administrador), onde cada perfil possui permissões específicas, sendo o administrador responsável pelo gerenciamento do catálogo e do estoque.

Do ponto de vista técnico, o sistema foi desenvolvido utilizando Programação Orientada a Objetos, com uma estrutura modular bem organizada. O código foi pensado para ser fácil de entender e manter, permitindo que novas funcionalidades sejam adicionadas sem grandes dificuldades, além de manter uma divisão clara das responsabilidades dentro do sistema.
